== Description ==

Astro WooCommerce Free Gift plugin is an extension of WooCommerce plugin that allow you to create a list of free Gift for any product of your shop.

== Frequently Asked Questions ==

= How to select product gifts? =

In edit product page > Product data box. Open Linked products tab and select some products for Gift product items.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate Astro WooCommerce Free Gift from Plugins page
